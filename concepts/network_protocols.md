# Network Protocols

#### Four layers of the Internet Protocol Stack

1. Application Layer
2. Transport Layer
3. Internet Layer
4. Link Layer

Link Layer:
- Responsible for sending data over a physical medium, such as a cable or a wireless connection.
- Examples: Ethernet, Wi-Fi

Internet Layer:
- Next layer up from the Link Layer, responsible for routing data across networks.
- To get data to right destination.
- Examples: IP (Internet Protocol)

Transport Layer:
- This layer's job is to provide reliable data transfer between two devices.
- Transport layer relies on services provided by internet layer.
- Examples: TCP (Transmission Control Protocol), UDP (User Datagram Protocol)

Application Layer:
- This is the top most layer, where most of the actual applications that we use live.
- The application layer relies on protocols like TCP/UDP in the Transport layer. 
- Examples: HTTP (Hypertext Transfer Protocol), FTP (File Transfer Protocol), SMTP (Simple Mail Transfer Protocol)
